# Sobre o Projeto
B2 Management será uma aplicação que se integrará ao serviço backblaze para gerir objetos na nuvem.

## Funcionalidades
- Usuários: Aplicação terá logins para ter acesso ao sistema com logs e permissões definidas
- Interface web para gestão do sistema
- Backend (API) com conexão ao banco de dados
- Listar buckets, objetos e versões de objetos
- Listar configurações de lifecycle do bucket
- Download e upload de arquivos para o bucket
- Criar, deletar e configurar buckets

## Stack tecnológica
### API
- Golang
- Gin
- Postgres
- Migrations
- Swagger
- Container

### Frontend
- React
- Next
- Typescript
