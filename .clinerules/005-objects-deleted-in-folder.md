# Visualizar objetos deletados
Quando um objeto foi deletado de dentro de uma pasta ele fica com a versão anterior invisivel.

### Implementação
- No modal de objetos, no rodapé, ao lado do do botão upload de arquivos, adicionar um checkbox com aopção Exibir Objetos excluidos.
- Quando este checkbox for marcado (true) deverá ser exibido o objeto que foi deletado.
- Exiba a versão deste objeto mais recente um pouco mais apagado e quando clicado segue a mesma forma de objetos atuais. Quando clicado é exibido todas as versões restantes.
